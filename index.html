Build AU · Feedback Widget v2

Prototype Feedback Widget

A lightweight, drop-in feedback overlay for Build AU prototypes. Reviewers place numbered pins anywhere on screen and leave comments. You review and resolve them from a password-protected panel.

Add to any prototype — one line
<script src="https://build-au-feedback-widget.vercel.app/feedback-widget.js"></script>

Place this just before the closing </body> tag of any prototype HTML file.
